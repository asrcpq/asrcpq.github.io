My projects:

* [tttz](https://github.com/asrcpq/tttz), a terminal tile-matching game with network battle.
[demo](https://asrcpq.github.io/resources/2107/tttz_demo.mp4).

* psva(source code not available), experimental physics engine
for deformation and destruction simulation based on XPBD.
[demo](https://github.com/asrcpq/asrcpq.github.io/tree/master/resources/psva_demo)

* [eyhv](https://github.com/asrcpq/eyhv), an experimental, high-speed, minimalist, flexible shmup
with a builtin software renderer.
[demo](https://asrcpq.github.io/resources/2005/eyhv_demo.gif)

* zsh plugins and UNIX desktop/shell utilities:
[a prompt](https://github.com/asrcpq/tlvprompt),
[autosuggestion alternative](https://github.com/asrcpq/myzshsug),
[autojump alternative](https://github.com/asrcpq/myjump),
[history merge tool](https://github.com/asrcpq/zsh_history_manager),
[natural sort](https://github.com/asrcpq/smartsort),
[i3status alternative](https://github.com/asrcpq/mystatus),
[terminal input method](https://github.com/asrcpq/myinput),
etc.
